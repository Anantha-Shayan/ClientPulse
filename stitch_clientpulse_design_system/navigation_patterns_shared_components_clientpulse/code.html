4. TabBar Pattern

Active State Logic

Tabs utilize a #1B4FE8 2px bottom border with font-bold for active states. Hover states use a subtle text color transition without structural movement.

Desktop Spacing

Horizontal padding is set to 24px (lg) with a 16px (md) bottom padding to maintain a comfortable visual rhythm above the container border.

Visual Hierarchy

Unselected tabs use Neutral 500 (on-surface-variant) to recede into the background, ensuring primary focus remains on active content.
